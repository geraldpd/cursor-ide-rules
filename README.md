## Code Style
-Always use vuetify for styling, minimal to no custom styles should be used.
-Always use pinia for state management.
-Always use vue-router for routing.
-Always add new lines between functions and methods.
-Always add new linesif conditions
-Always group codes where it makes sense to do so.
-Always use descriptive names for variables and functions.
